Olá. Vamos explicar brevemente
como funciona o Aleatorium! :D
Nós geramos um valor aleatório
entre 0 e 99. O botão ( 00 ~ 99 )
reinicia esse valor. O Botão Play
( ► ) testa o número desejado e
contabiliza o tanto de tentativas,
além de lhe informar no visor se
a resposta é maior ou menor que
o número que você tentou. EX:
Você tentou 25, e a resposta é 78
No visor aparecerá: MAIOR (>).


Ah, o número embaixo do chute
são quantas vezes você tentou. :D
